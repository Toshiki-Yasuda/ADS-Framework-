## ⚙️ ワークフロー比較

### シナリオ: READMEのタイポ修正

#### Full版のワークフロー

```
ユーザー: "READMEのタイポを修正して"
    ↓
1. ADS層判定 (CLAUDE.md自動読み込み)
   → 「これは実装タスクか？エスカレーションか？」
   → 判定: 実装で進める
   ⏱️ 5秒

2. /project:aom-plan "READMEタイポ修正"
   → タスク分解実行
   → TASK-001: README.md の修正
   → 依存関係グラフ（DAG）構築
   ⏱️ 20秒

3. /project:agm-branch "docs/fix-readme-typo"
   → ブランチ作成: docs/fix-readme-typo
   ⏱️ 5秒

4. /project:eam-implement "TASK-001"
   → README.md を読み込み
   → タイポ箇所を特定
   → Edit ツールで修正
   ⏱️ 30秒

5. /project:agm-commit "auto"
   → git diff で変更確認
   → コミットメッセージ自動生成
   → コミット実行: "docs: fix typo in README"
   ⏱️ 10秒

合計時間: 約 70秒
コマンド実行回数: 4回
```

#### Solo Edition のワークフロー

```
ユーザー: "READMEのタイポを修正して"
    ↓
1. CLAUDE.md 自動読み込み
   → 開発ワークフローを理解
   ⏱️ 0秒（バックグラウンド）

2. 直接実行
   → README.md を読み込み
   → タイポ箇所を特定
   → Edit ツールで修正
   → git add README.md
   → git commit -m "docs: fix typo in README"
   ⏱️ 15秒

合計時間: 約 15秒
コマンド実行回数: 0回（直接実行）
```

**効率化**: 70秒 → 15秒 = **79%短縮**

## 🚀 使い分けガイド

### Full版を選ぶべきケース

| 条件 | 理由 |
|------|------|
| **チーム開発（5人以上）** | 役割分担・レビュープロセスの明確化が必要 |
| **大規模プロジェクト（10,000行以上）** | 複雑なタスク管理・依存関係の可視化が必要 |
| **長期運用（6ヶ月以上）** | 厳格なワークフローの維持が重要 |
| **品質基準が厳格** | 多層チェック・エスカレーションフローが有効 |
| **Git操作の自動化重視** | 9個のコマンドでGit操作を完全自動化 |
| **新規参加者が頻繁** | 体系的なオンボーディングプロセスが必要 |

**具体例**:
- エンタープライズアプリケーション
- マイクロサービス群の管理
- オープンソースプロジェクト（多数のコントリビューター）

---

### Solo Edition を選ぶべきケース

| 条件 | 理由 |
|------|------|
| **個人開発（1-2人）** | シンプルさが最優先 |
| **小〜中規模プロジェクト（〜10,000行）** | 複雑な階層は不要 |
| **短期〜中期（〜6ヶ月）** | 迅速な開発を重視 |
| **プロトタイプ・MVP開発** | スピード優先 |
| **学習目的・趣味プロジェクト** | 学習コストを最小化 |
| **Claude Code初心者** | まずは軽量版から始める |

**具体例**:
- 個人ブログ
- Todoアプリ
- APIクライアントライブラリ
- サイドプロジェクト

---

## 📊 まとめ: どちらを選ぶか？

### 決定フローチャート

```
プロジェクト開始
    |
    ├─ チーム人数は？
    │   ├─ 1-2人 → Solo Edition
    │   └─ 5人以上 → Full版
    |
    ├─ プロジェクト規模は？
    │   ├─ 〜10,000行 → Solo Edition
    │   └─ 10,000行以上 → Full版
    |
    ├─ 開発期間は？
    │   ├─ 〜6ヶ月 → Solo Edition
    │   └─ 6ヶ月以上 → Full版
    |
    └─ 重視するものは？
        ├─ スピード・シンプルさ → Solo Edition
        └─ 厳格な品質管理 → Full版
```

### 最終推奨

| あなたの状況 | 推奨版 |
|------------|--------|
| 個人開発、小規模、初心者 | **Solo Edition** |
| 個人開発、中規模、経験者 | Solo Edition または Solo + Skills |
| 小チーム（2-4人）、中規模 | Solo + Skills または Light Full版 |
| 大チーム（5人以上）、大規模 | **Full版** |
